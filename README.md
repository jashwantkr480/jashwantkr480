### 🚀 Featured Projects

#### 📲Social Media Privacy & Security Trade-off Analysis (Python, Reddit API, NLP, Thematic Coding)
*Python · NLP · Graph Analysis · Privacy Modeling · Sentiment Analysis*

Engineered a large-scale data pipeline collecting **30k + posts across 11 subreddits and 25 security-focused keywords**, and designed a **4-label inter-rater reliability framework (Cohen's κ ≥ 0.70)** to systematically classify user-reported threats, protective behaviors, and unresolved vulnerabilities.

Surfaced a **1.7:1 threat-to-mitigation asymmetry** and cross-community framing divergence revealing a structural privacy self-defense gap among general social media users — with implications for platform accountability and digital safety policy.

#### 🛣️ Road Safety Analytics & Graph-Based Routing
*Python · DBSCAN · OpenStreetMap · Folium*

Analyzed **7M+ U.S. accident records** across 5 California cities to engineer a weighted risk model. Built a routing engine that generates safer alternative routes within a 500m radius and visualizes accident hotspots interactively.

#### 💬 Cross-Platform Sentiment Drift Analysis
*BERT · RoBERTa · GraphSAGE · Node2Vec · LDA · BERTopic*

Built a multi-platform NLP pipeline using VADER, BERT, and RoBERTa across **Twitter, Reddit, and YouTube**. Modeled sentiment propagation and temporal drift using graph ML and topic modeling to track narrative shifts over time.

#### 🏥 2D-to-3D Medical Image Reconstruction
*U-Net · DenseDepth · Depth Estimation · Mobile UI*

Converted 2D medical scans into interactive 3D models using monocular depth estimation. Enabled real-time visualization and manipulation to improve diagnostic accessibility for clinicians.

#### 🏎️ Formula 1 Race Strategy & Championship Analysis Dashboard
*Python · Plotly · Streamlit · FastF1 API · Interactive Visualization*

Built an interactive analytics platform exploring the 2021 Formula 1 championship controversy, visualizing lap times, pit strategies, safety-car windows, and driver position changes using official telemetry data from the FastF1 API. The dashboard enables users to interactively analyze how race events and FIA decisions influenced the championship outcome through dynamic charts and scenario visualizations.
